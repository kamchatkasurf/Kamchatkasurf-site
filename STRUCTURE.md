# Структура сайта «Камчатка Серф»

```
Новая папка/
├── index.html              # Главная страница (лендинг)
├── css/
│   └── styles.css          # Стили, шрифты, адаптив
├── js/
│   └── main.js             # Карусели, попапы, форма, lightbox, CTA
├── legal/
│   ├── privacy.html        # Редирект → Политика конфиденциальности
│   ├── offer.html          # Редирект → Договор оферты
│   ├── terms.html          # Редирект → Пользовательское соглашение
│   ├── personal-data.html  # Редирект → Согласие на обработку данных
│   └── [документы .html]   # Полные тексты юридических документов
├── assets:fonts/           # Шрифты (Gogono Cocoa Mochi, Roboto Mono)
├── assets:icons/           # Иконки
├── assets:images/          # Фото (hero-mobile, карусель, программы, галерея)
│   └── gallery/            # Фото для блока «Жизнь лагеря»
├── assets:logo/            # Logo.svg
├── assets:video/           # hero.mov / hero.mp4 для Hero
└── STRUCTURE.md            # Этот файл
```

## Блоки на странице

1. **Hero** — видеофон (на мобильных — hero-mobile.jpg), логотип, заголовок
2. **Social proof** — 4 карточки с цифрами и count-up
3. **Философия** — карусель «Серфинг на Камчатке» (4 слайда)
4. **Наши программы** — 4 карточки с попапами
5. **Отзывы** — карусель отзывов (стрелки + точки)
6. **Фотогалерея** — сетка фото + lightbox
7. **Форма заявки** — POST на /api/lead
8. **Footer** — ссылки на юридические документы, реквизиты

## CTA

Фиксированная кнопка «Хочу в серф-лагерь» (правый нижний угол) → плавный скролл к форме.
