# Настройка отправки заявок в Telegram

Заявки с формы будут приходить в ваш Telegram. Нужно выполнить 3 шага.

## 1. Создать бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: Kamchatka Surf Bot)
4. Введите username бота (например: kamchatka_surf_bot)
5. **Скопируйте токен** — длинная строка вида `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

## 2. Узнать свой Chat ID

1. Напишите вашему новому боту любое сообщение (например: `/start`)
2. Откройте в браузере:  
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`  
   (подставьте токен вместо `<ВАШ_ТОКЕН>`)
3. В ответе найдите `"chat":{"id":123456789}` — это ваш **chat_id**

## 3. Настроить переменные при деплое

### Если используете Vercel

1. Загрузите проект на [vercel.com](https://vercel.com)
2. Откройте проект → **Settings** → **Environment Variables**
3. Добавьте:
   - `TELEGRAM_BOT_TOKEN` = токен от BotFather
   - `TELEGRAM_CHAT_ID` = ваш chat_id (только число)

4. Сделайте **Redeploy** проекта

### Если используете другой хостинг

Нужен Node.js сервер, который будет вызывать Telegram API. Файл `api/lead.js` рассчитан на Vercel Serverless. Для других платформ (Netlify, свой сервер) потребуется адаптация.
